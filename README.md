# Sports Data Analysis Project

## Description
This project analyzes football match data to identify scoring trends,
home vs away performance, and general match outcomes.

## Tools Used
- Python
- Pandas
- CSV datasets

## Key Insights
- Average goals per match
- Comparison of home wins, away wins, and draws
- Identification of goal-scoring trends

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run: python analysis.py

## Why This Project Matters
Sports data analysis is essential in betting platforms for understanding
probability trends, risk assessment, and match outcome patterns.
